Projects

Diffusion-Based Forecasting for Uncertainty-Aware Model Predictive Control

Diffusion-Informed MPC is a framework that integrates diffusion-based probabilistic forecasting into model predictive control for decision-making in partially observable stochastic systems. Tested on battery energy arbitrage in the New York day-ahead electricity market; outperforms classical forecasters and model-free RL baselines.

IEEE EUSIPCO 2025 (accepted) • Official PDF • Code

Diffusion-Based Scenario Tree Generation for Multivariate Time Series Prediction and Multistage Stochastic Optimization

Diffusion Scenario Tree (DST) is a method for constructing scenario trees using diffusion forecasters. Tested on battery energy arbitrage in the New York day-ahead electricity market; outperforms VAR/LSTM-based trees, standard MPC with diffusion and classical forecasters, and model-free RL.

Submitted to IEEE ICASSP 2026 • arXiv:2509.14832 • Code

Diploma Thesis: Decision-Making in Stochastic Environments Using Diffusion Models

Developed Diffusion-Informed Model Predictive Control (D–I MPC), a unified framework that integrates diffusion-based probabilistic forecasting into several MPC variants (deterministic, stochastic, scenario-tree, and heuristic-augmented) for sequential decision-making under uncertainty. Applied to battery energy arbitrage in the New York day-ahead electricity market; consistently outperforms classical forecasters, model-free RL baselines, and closely matches idealized perfect-forecast benchmarks.

M.Eng. Diploma Thesis, School of ECE, NTUA • June 2025 • Grade 10/10 • Thesis PDF • Code

Baby Sophia: A Developmental Approach to Self-Exploration through Self-Touch and Hand Regard

Curiosity-driven RL framework for a simulated infant robot that learns self-touch and hand regard using only intrinsic rewards. Uses semantic body map compression and multi-component curiosity rewards + curricula to reproduce infant developmental patterns.

For the IEEE ICDL 2025 BabyBench Competition • arXiv:2511.09727 • Code
